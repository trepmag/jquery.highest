jQuery Highest

jQuery Highest is a jQuery plugin which set the height of all matching elements to the highest found.

Example

In this example the script is setup with the {window_min_width: 845} option which tell the plugin to stop applying the height to the highest when the window size is lesser than 845px (Try resizing the window to see this behavior).
